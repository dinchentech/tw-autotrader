# 預先準備（一次性設定）

## Docker 免 sudo

WSL 內建的 Docker daemon 需要 `sudo` 才能操作。避免每次都要打 `sudo`，把自己加入 `docker` group：

```bash
sudo usermod -aG docker $USER
```

> ⚠️ **生效條件**：執行後需**登出 WSL 再重新打開終端機**才會生效。
> 可以用 `groups` 指令確認是否已加入 docker group。

## WSL Docker Desktop 整合（替代方案）

如果 Windows 有裝 Docker Desktop，啟用 WSL2 整合也可以免 sudo：

1. 打開 Docker Desktop → Settings → Resources → WSL Integration
2. 啟用你的 WSL 發行版（Ubuntu）
3. Save & Restart
4. 重新打開 WSL 終端機

之後 `docker` 指令就不需要 `sudo` 了。

## GCP gcloud 認證

如果 `gcloud` 指令需要重新認證：

```bash
gcloud auth login
gcloud auth configure-docker asia-east1-docker.pkg.dev
```
