## 核心觀念
1. **if / elif / else**：如果（if）條件成立做 A，否則如果（elif）做 B，再否則（else）做 C。
2. **比較運算子**：`>`（大於）、`<`（小於）、`>=`（大於等於）、`<=`（小於等於）、`==`（等於）、`!=`（不等於）。
3. **and（且）**：兩個條件都要成立，整題才成立。
4. **or（或）**：兩個條件只要一個成立，整題就成立。
5. **這些就是策略的核心**：所有策略的買賣決定，到最後就是一個或多個條件的組合。
