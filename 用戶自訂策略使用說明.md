# 用戶自訂策略使用說明

## 概述

本系統提供四組備用策略供您自由定義，無需修改加密的主程式 `live_trader_multi.py`。

### 策略分類

| 策略代號 | 用途 | 位置 |
|---------|------|------|
| `g1_strategy_1` | Group 1 備用策略 1 | `user_strategies.py` → `my_custom_ma_cross()` |
| `g1_strategy_2` | Group 1 備用策略 2 | `user_strategies.py` → `my_custom_rsi()` |
| `g2_strategy_1` | Group 2 備用策略 1 | `user_strategies.py` → `my_custom_momentum()` |
| `g2_strategy_2` | Group 2 備用策略 2 | `user_strategies.py` → `my_custom_volume_price()` |

## 快速開始

### 1. 修改 `.env` 設定

在 `.env` 檔案中添加：

```bash
# Group 1 個股設定（使用備用策略）
PC_0050='{"strategy": "g1_strategy_1", "alloc": 20}'
PC_2330='{"strategy": "g1_strategy_2", "alloc": 30}'

# Group 2 備用策略（如需要，可搭配 INST_MOM_CAPITAL 使用）
# G2_STRATEGY_1=my_custom_momentum
# G2_STRATEGY_2=my_custom_volume_price
```

或使用舊格式：

```bash
PORTFOLIO=0050:g1_strategy_1,2330:g1_strategy_2
```

### 2. 自訂策略參數

使用 PC_ 格式可傳遞參數：

```bash
PC_0050='{"strategy": "g1_strategy_1", "alloc": 20, "fast_period": 10, "slow_period": 20}'
PC_2330='{"strategy": "g1_strategy_2", "alloc": 30, "rsi_period": 14, "oversold": 25, "overbought": 75}'
```

### G1 Strategy 1: my_custom_ma_cross (均線策略)

**預設邏輯**：簡單均線交叉策略
- 快線穿過慢線向上 = 買進
- 快線穿過慢線向下 = 賣出

**參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `fast_period` | 5 | 快線週期 |
| `slow_period` | 10 | 慢線週期 |

### G1 Strategy 2: my_custom_rsi (RSI 策略)

**預設邏輯**：RSI 超買超賣策略
- RSI < 30 = 超賣，買進
- RSI > 70 = 超買，賣出

**參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `rsi_period` | 14 | RSI 計算週期 |
| `oversold` | 30 | 超賣門檻 |
| `overbought` | 70 | 超買門檻 |

### G2 Strategy 1: my_custom_momentum (動能策略)

**預設邏輯**：價格動能策略
- 過去 N 日漲跌幅超過門檻 = 買進
- 過去 N 日跌幅超過門檻 = 賣出

**參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `lookback` | 5 | 回看天數 |
| `threshold` | 3 | 漲跌幅門檻（百分比） |

---

### G2 Strategy 2: my_custom_volume_price (量價策略)

**預設邏輯**：量價配合策略
- 價格站穩均線 + 成交量放大 = 買進
- 價格跌破均線 = 賣出

**參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `ma_period` | 10 | 價格均線週期 |
| `volume_ma_period` | 10 | 成交量均線週期 |
| `volume_mult` | 1.5 | 成交量放大倍數 |

## 如何自訂策略

### 步驟 1：開啟 `user_strategies.py`

```bash
vim user_strategies.py
```

### 步驟 2：修改策略函式

每個策略函式都需要遵循以下格式：

```python
import pandas as pd

def my_strategy_name(df: pd.DataFrame, **kwargs) -> pd.DataFrame:
    """
    策略說明

    Parameters:
    - df: 包含 OHLCV 資料的 DataFrame
        - open: 開盤價
        - high: 最高價
        - low: 最低價
        - close: 收盤價
        - volume: 成交量
    - kwargs: 策略參數

    Returns:
    - DataFrame with 'signal' column
        - 1 = BUY (買進)
        - -1 = SELL (賣出)
        - 0 = HOLD (不動作)
    """
    # 您的策略邏輯
    signals = pd.Series(0, index=df.index)

    # ... 計算訊號 ...

    return pd.DataFrame({'signal': signals}, index=df.index)
```

### 步驟 3：在 `USER_STRATEGY_MAP` 中註冊

```python
USER_STRATEGY_MAP = {
    'your_strategy_name': my_strategy_name,
}
```

### 步驟 4：在 `.env` 中使用

```bash
PC_2330='{"strategy": "your_strategy_name", "alloc": 20}'
```

## 注意事項

1. **必須回傳 DataFrame**：策略函式必須回傳包含 `signal` 欄位的 DataFrame
2. **訊號值**：`signal` 只能是 `1` (BUY)、`-1` (SELL) 或 `0` (HOLD)
3. **索引對齊**：回傳的 DataFrame 索引必須與輸入 `df` 相同
4. **參數傳遞**：透過 `.env` 中的 PC_ 設定傳遞參數

## 常見問題

### Q: 如何測試我的策略？

A: 可以使用回測功能驗證策略：
```bash
python backtest.py --strategy your_strategy_name
```

### Q: 可以同時使用內建策略和自訂策略嗎？

A: 可以！內建策略包括：
- `bollinger` - 布林通道
- `ma_cross` - 均線交叉
- `vwap` - VWAP 偏離
- `breakout` - 突破策略
- `keep_wait` - DCA 低接

搭配範例：
```bash
PC_0050='{"strategy": "bollinger", "alloc": 30}'
PC_2330='{"strategy": "g1_strategy_1", "alloc": 20}'
```

### Q: 策略執行出錯怎麼辦？

A: 檢查 `logs/` 目錄下的日誌檔案，錯誤訊息會顯示具體問題。

## 範例：完整設定流程

### 1. 自訂一個簡單策略

在 `user_strategies.py` 中添加：

```python
def my_simple_strategy(df: pd.DataFrame, **kwargs) -> pd.DataFrame:
    """簡單策略：收盤價高於開盤價買進"""
    signals = pd.Series(0, index=df.index)
    signals[df['close'] > df['open']] = 1
    signals[df['close'] < df['open']] = -1
    return pd.DataFrame({'signal': signals}, index=df.index)

USER_STRATEGY_MAP['my_simple'] = my_simple_strategy
```

### 2. 在 `.env` 中使用

```bash
PC_0050='{"strategy": "my_simple", "alloc": 20}'
```

### 3. 啟動系統

```bash
python live_trader_multi.py
```
