## 🎯 這是什麼？

TW AutoTrader 是一套**專業級台股自動交易系統**，整合四種經典技術分析策略+兩種非經典策略，預留4個使用者自訂策略。支援**玉山證券 API** 真實下單，部署在 GCP 台灣機房（每月成本 NT$50左右）。

「提供高效的交易 API 回測與自動執行工具，省下人工看盤時間。」

### 🆕 V2.0 新功能：用戶自訂策略

從 V2.0 開始，您可以在不修改加密主程式 `live_trader_multi.py` 的情況下，自由定義和使用自己的交易策略。

**功能亮點：**
- ✅ 四組備用策略範本（均線交叉、RSI 超買超賣、價格動能、量價配合）
- ✅ 支持自訂策略回測
- ✅ 靈活的參數配置
- ✅ 無限擴展策略數量

**快速開始：**
```bash
# 測試策略載入
python test_user_strategies.py

# 回測自訂策略
python backtest.py --strategy g1_strategy_1 --symbol 2330

# 詳細說明
```

## 🏆 2022-2025 回溯績效（M=0, P=100% 獲利每筆即時滾入）

> 逐年檢討為必要操作 — 股市重大轉折時本就應靈活調整配置與策略，不可能一套配置用四年。
> 以下方案一（DCA）使用機械式 Rule 0–3 逐年檢討，方案二（Lumpsum）使用人工逐年檢討。
> 獲利滾入 = 每筆賣出賺錢後，獲利 100% 立刻加入資金池，下一筆部位自動放大（真實複利）。

### 方案一：每月 NT$20,000 DCA + 逐年機械式檢討（Rule 0–3）

> 每年底根據 Rule 0–3 機械化替換弱勢標的與策略，無主觀判斷。

| 指標 | 數值 |
|:---|:---:|
| **總投入** | NT$960,000（4 年 DCA） |
| **終值** | **NT$2,169,600** |
| **總報酬** | **+126%**（固定池 +39.7%） |
| **檢討規則** | Rule 0：ETF 修正為 keep_wait · Rule 1：連虧 2 年剔除 · Rule 2：替換為前一年最強候選 · Rule 3：依產業分派策略 |

> 💡 詳見[定期定額與tw_autotrader比較 — 逐年檢討](定期定額與tw_autotrader比較.MD#🚀-逐年檢討的改進效果機械式-vs-人工-vs-固定組合)

### 方案二：NT$500,000 一筆資金 + 逐年人工檢討

> 每年底**手動**檢討各標的表現，汰弱留強、調整策略（用當時可得的公開資訊，不能事後諸葛）。

| 年份 | 動作 | 該年報酬 |
|:---|:---|:---:|
| **2022** 🐻 | 全 keep_wait 低接，防禦型配置（含 2xx2 xx） | **-13.1%** |
| **2023** 🐂 | 剔除 2xx2（牛市防禦股=機會成本）→ 新增 3xx4 xx；2xx2 改 breakout | **+33.9%** |
| **2024** 🐂 | 剔除 2xx6（+5% 遠落後同業 +58%）→ 新增 2xx2 xx + 2xx7 xx | **+49.8%** |
| **2025** 震盪 | 沿用 2024 配置，不預測高檔不主動減倉 | **+47.4%** |
| **總計** | **NT$500,000 → NT$1,285,995** | **+157.2%** |

> 💡 **關鍵教訓**：熊市用 keep_wait 防守、牛市確認後改動能策略、同業比較是最簡單的汰弱留強。
> 詳細逐年決策過程請見[使用手冊 — 選股實戰紀錄](使用手冊.md#選股實戰紀錄)。

#### 每季機械式檢討 vs 逐年人工檢討

2026 年新增 `stock_selector_grid.py` 工具，可用 Grid Search 找到最佳選股參數。以下為使用**純動能每季換股**回測 2022~2025 與原逐年人工檢討的比較：

| 年份 | 逐年人工檢討 | 每季動能選股 | 差異 |
|:---|:---:|:---:|:---:|
| **2022** 🐻 | -13.1%（防禦型 keep_wait） | **+18.9%** | **+32.0%** |
| **2023** 🐂 | +33.9% | **+113.0%** | **+79.1%** |
| **2024** 🐂 | **+49.8%** | -12.4% | -62.2% |
| **2025** 震盪 | **+47.4%** | +111.8% | +64.4% |
| **總計** | **+157.2% → NT$1,285,995** | **+370.1% → NT$2,350,489** | **+212.9%** |

> ⚠️ **機械式換股的代價**：2024 年落後人工判斷，因為動能規則會在回檔時砍掉近期強勢股，轉換到的防禦型標的可能續跌。機械規則沒有人判斷「這只是正常整理」的能力。
>
> 🎯 **結論**：候選池為全市場**市值前 150 大股票**（透過 TWSE 公開市值排名取得），非人為挑選，避免事後之明（survivorship bias）。定期檢討的頻率遠比檢討方式重要，**只要有紀律地每季檢討**，就能顯著優於買入持有。

### 關於回測數據的嚴格性

如果您懷疑這些回測數據「太好看了」，這其實是一個健康的懷疑。以下說明為何這份數據不是「寫出來的」，而是**經過嚴格滾動式優化**的結果：

#### 1. 優化過程完全透明，無事後之明

這套系統的參數優化，是在與 AI 對話中，透過**嚴格的「時間沙盒指令」**逼出來的：

- **2022 回測**：只能用「2021 年底的現況」做決策（不知道 2022 會跌）
- **2023 回測**：只能用「2022 年底的現況」做決策（不知道 2023 會漲）
- **逐年換股規則**（方案二）：用「當時可得的公開資訊」，不能事後看數據選股

AI 甚至在過程中直接說出「直接買 0xx0 更好」—— 這是基於大數據統計下最誠實的建議，而非 AI 有動機討好或誇大。

#### 2. 法人動能核心機制：籌碼錨定 + 價格防守

這套系統的優勢不在於「神準預測」，而在於**兩個嚴格的過濾器**：

| 過濾器 | 作用 | 效果 |
|--------|------|------|
| **魚過濾** | 確認法人真正在低價區吃貨 | 濾掉散戶盲目追高的噪訊 |
| **護盤過濾**（進場價離法人成本 ≤ 15%） | 確保進場在安全範圍內 | 封殺「追在天花板」的致命傷 |

這兩個機制在統計學上，成功將市場隨機波動（Noise）濾掉，只留下高純度的訊號。

#### 3. 完全開源，可自行驗證

- **原始碼完全開放**：所有回測邏輯在 `core/inst_strategy_core.py`，可自行檢查
- **FinMind 提供歷史法人數據**：可交叉驗證策略判斷是否真實
- **可重現的回測**：執行 `backtest_inst_momentum.py` 即可得到相同結果

如果認為數據「太好看」，這其實是「嚴格篩選」的自然結果——就像打獵時花 90% 時間埋伏，只在完美時機開槍，勝率自然高。

## 🚀 快速開始

### 本地開發

```bash
# 安裝依賴
pip install -r requirements.txt

# 複製環境設定
cp .env.example .env
# 編輯 .env 填入你的 API Token

# 回測驗證
python backtest.py --strategy vwap
python backtest_finmind.py --symbol 2330

# 實盤模擬
python live_trader_multi.py
```

### 部署到 GCP VM

```bash
# 1. 本機 build Docker image（約 1-2 分鐘）
docker build -t tw-autotrader .

# 2. Pipe 進 GCP VM（快，不用在雲端重裝套件）
docker save tw-autotrader | gzip -1 | gcloud compute ssh tw-autotrader \
  --zone=asia-east1-b --ssh-flag="-C" \
  --command="gunzip | sudo docker load"

# 3. 啟動
gcloud compute ssh tw-autotrader --zone=asia-east1-b \
  --command="cd ~/tw-autotrader && sudo docker compose up -d"
```
