# TW AutoTrader — 台股自動交易系統

📖 [完整使用手冊](使用手冊.md) · 📊 [策略說明](策略說明.md) · 🔙 [回溯說明](回溯說明.MD)



## 🎯 這是什麼？

TW AutoTrader 是一套**專業級台股自動交易系統**，整合四種經典技術分析策略，支援**玉山證券 API** 真實下單，部署在 GCP 台灣機房（每月成本 ~NT$50）。

## 🏆 2022-2025 回溯績效（M=0, P=100% 獲利每筆即時滾入）

> 逐年檢討為必要操作 — 股市重大轉折時本就應靈活調整配置與策略，不可能一套配置用四年。
> 以下方案一（DCA）使用機械式 Rule 0–3 逐年檢討，方案二（Lumpsum）使用人工逐年檢討。
> 獲利滾入 = 每筆賣出賺錢後，獲利 100% 立刻加入資金池，下一筆部位自動放大（真實複利）。

### 方案一：每月 NT$20,000 DCA + 逐年機械式檢討（Rule 0–3）

> 每年底根據 Rule 0–3 機械化替換弱勢標的與策略，無主觀判斷。
> 回測腳本：`python backtest_yearly_review.py`

| 指標 | 數值 |
|:---|:---:|
| **總投入** | NT$960,000（4 年 DCA） |
| **終值** | **NT$2,169,600** |
| **總報酬** | **+126%**（固定池 +39.7%） |
| **檢討規則** | Rule 0：ETF 修正為 keep_wait · Rule 1：連虧 2 年剔除 · Rule 2：替換為前一年最強候選 · Rule 3：依產業分派策略 |

> 💡 詳見[定期定額與tw_autotrader比較 — 逐年檢討](定期定額與tw_autotrader比較.MD#🚀-逐年檢討的改進效果機械式-vs-人工-vs-固定組合)

### 方案二：NT$500,000 一筆資金 + 逐年人工檢討

> 每年底**手動**檢討各標的表現，汰弱留強、調整策略（用當時可得的公開資訊，不能事後諸葛）。
> 回測腳本：`python backtest_manual_review.py`

| 年份 | 動作 | 該年報酬 |
|:---|:---|:---:|
| **2022** 🐻 | 全 keep_wait 低接，防禦型配置（含 2412 中華電） | **-13.1%** |
| **2023** 🐂 | 剔除 2412（牛市防禦股=機會成本）→ 新增 3034 聯詠；2382 改 breakout | **+33.9%** |
| **2024** 🐂 | 剔除 2886（+5% 遠落後同業 +58%）→ 新增 2882 國泰金 + 2317 鴻海 | **+49.8%** |
| **2025** 震盪 | 沿用 2024 配置，不預測高檔不主動減倉 | **+47.4%** |
| **總計** | **NT$500,000 → NT$1,285,995** | **+157.2%** |

> 💡 **關鍵教訓**：熊市用 keep_wait 防守、牛市確認後改動能策略、同業比較是最簡單的汰弱留強。
> 詳細逐年決策過程請見[使用手冊 — 選股實戰紀錄](使用手冊.md#-選股實戰紀錄20222025-逐年檢討模擬)。

### 方案三：NT$500,000 法人抬轎動能策略（🐟 魚過濾 每日篩選 + 法人動能 + 護盤過濾）

| 指標 | 數值 |
|------|------|
| **策略** | 每日篩選，兩階段：①魚過濾（法人低吃分數 ≥ 7.0，回溯 90 天）②動能進場（創新高 + 站穩 MA + 法人買超）③護盤過濾（進場價離法人成本 ≤ 15%） |
| **回測期間** | 2022-01-01 → 2025-12-31（4 年完整牛熊） |
| **起始本金** | NT$500,000 |
| **持有檔數** | TOP_N=3（固定，隨資金池放大部位） |
| **停損停利** | -10% 硬性停損 / MA10 移動停利 |
| **總交易** | 198 筆 |
| **勝率** | 71.72% |
| **最大回撤** | 22.02%（NT$167,780） |
| **最終權益** | NT$926,120（+85.22%，年化 +16.7%） |
| **獲利滾入** | M=0, P=100%（每筆贖回即時滾入，真實複利） |

| 年份 | 年初權益 → 年底權益 | 該年報酬 | 平均動用 | 最高動用 |
|------|---------------------|---------|---------|---------|
| **2022** 🐻 | NT$500,000 → NT$469,269 | **-6.1%** | — | — |
| **2023** 🐂 | NT$469,269 → NT$526,544 | **+12.2%** | — | — |
| **2024** 🐂 | NT$526,544 → NT$693,174 | **+31.6%** | — | — |
| **2025** 🐂 | NT$693,174 → NT$926,120 | **+33.6%** | — | — |

> 💡 **護盤過濾（INST_MOM_MAX_DIST_FROM_ACCUM=0.15）**：進場價離法人成本超過 15% 即視為護盤而非逢低布局，剔除後績效從 +67% 提升至 +85%，勝率從 67% 提升至 72%。
> 💡 **資金效率**：平均動用資金 NT$43 萬（86%），最高 NT$96 萬。
> 💡 獲利滾入（M=0, P=100%）複利效果顯著：+85% vs P=0%（固定資金）僅 +29.86%。
> 💡 魚過濾（score ≥ 7.0, days=90）為 Grid Search 最佳參數，STOCK_NO=150 為績效甜蜜點（搜尋150大市值股）。

## 🚀 快速開始

### 本地開發

```bash
# 安裝依賴
pip install -r requirements.txt

# 複製環境設定
cp .env.example .env
# 編輯 .env 填入你的 API Token

# 回測驗證
python backtest.py --strategy vwap
python backtest_esun.py --symbol 2330 --strategy vwap

# 實盤模擬
python live_trader_multi.py
```

### 部署到 GCP VM

```bash
# 1. 本機 build Docker image（約 1-2 分鐘）
docker build -t tw-autotrader .

# 2. Pipe 進 GCP VM（快，不用在雲端重裝套件）
docker save tw-autotrader | gzip -1 | gcloud compute ssh tw-autotrader \
  --zone=asia-east1-b --ssh-flag="-C" \
  --command="gunzip | sudo docker load"

# 3. 啟動
gcloud compute ssh tw-autotrader --zone=asia-east1-b \
  --command="cd ~/tw-autotrader && sudo docker compose up -d"
```
