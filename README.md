申請 LINE Notify 的免費權杖（Token）:
非常簡單，只需要 5 個步驟，完全不需要寫任何程式碼。請依照以下流程操作：
步驟 1：登入 LINE Notify 官方網站使用電腦瀏覽器打開 LINE Notify 官方網站 [1]。點擊網頁右上角的 「登入」。輸入您平時手機使用的 LINE 帳號與密碼進行登入。
步驟 2：進入個人頁面登入成功後，點擊網頁右上角您的 「LINE 帳號名稱」。在下拉選單中，點選 「個人頁面」。
步驟 3：發行權杖（Token）在個人頁面中往下捲動，找到「發行權杖」區塊。點擊 「發行權杖」 的綠色按鈕。
步驟 4：設定通知名稱與聊天室此時會彈出一個設定視窗，請填寫以下兩個欄位：填寫權杖名稱：請輸入發送通知時顯示的機器人名稱，例如輸入：台股策略機器人。選擇接收的聊天室：如果只想發給自己：請選擇清單最上方的 「透過1對1聊天接收LINE Notify的通知」。如果想發給群組（親友或合作夥伴）：請在清單中找到並勾選該 「LINE 群組」。填寫與選擇完畢後，點擊最下方的 「發行」 按鈕。
步驟 5：複製並妥善保存權杖畫面上會出現一串非常長的英文與數字組合，這就是您的 LINE Notify Token（權杖密碼）。點擊 「複製」 按鈕。
⚠️ 關鍵注意：這個密碼只會顯示這一次。請立刻把它貼到剛才討論的 DigitalOcean 主機 .env 檔案中，或是先儲存在您的電腦記事本裡。如果關閉這個視窗，您就再也看不到了，只能刪除重新申請。
🔔 最終啟動：如果是發送到「群組」的必做動作如果您在步驟 4 選擇的是將通知發送到某個 LINE 群組，發行完畢後，您必須手動打開手機的 LINE，進入該群組，將 「LINE Notify」 這個官方帳號邀請加入該群組。如果忘記邀請它進群，您的 Python 策略程式雖然執行成功，群組內也會收不到任何訊息。
