# 50萬一筆資金 — 台股自動交易配置

> ⚠️ **免責聲明**：本文僅為個人研究筆記，不構成任何投資建議。
> 自動化交易涉及市場風險，過去績效不代表未來獲利。

## 1. 配置總覽

這是**一筆資金**（NT$500,000）的配置方式，非每月定期定額。
初始資金依策略比例分配至各標的，獲利了結後資金回到池中可再投入。

| 策略 | 資金佔比 | 分配金額 | 標的 | 策略類型 |

| 布林通道反轉 | 40% | NT$200,000 | 0050, 006208, 00878 | 逆勢（均值回歸） |
| 均線交叉 | 25% | NT$125,000 | 2330, 2454 | 順勢（趨勢跟隨） |

## 2. 核心規則

1. **一筆投入，獲利再投**：初始500K一次性分配，停利後資金回歸策略池可再運用
2. **訊號驅動**：各策略依技術指標獨立產生買賣訊號，不預測市場
3. **資金效率**：未觸發訊號時資金保留現金，避免追高
4. **分散風險**：8檔標的 × 4策略，單一標的失效不影響整體
5. **入場過濾**：大盤年線過濾（指數<MA200時暫停買進）

## 3. .env 設定

```env
# === 基礎設定 === 
FINMIND_API_TOKEN=你的FinMindToken
INITIAL_CAPITAL=500000
MAX_RISK_PER_TRADE=0.02
MAX_DAILY_LOSS=0.03
MAX_DAILY_TRADES=5

# === 策略參數 ===
VWAP_SIGMA_MULT=1.5
VWAP_RSI_PERIOD=5
MA_CROSS_FAST_PERIOD=9
MA_CROSS_SLOW_PERIOD=21
MA_CROSS_ATR_THRESHOLD=0.005
BOLLINGER_WINDOW=20
BOLLINGER_STD_DEV=2.0
BOLLINGER_RSI_PERIOD=5
BREAKOUT_LOOKBACK=20
BREAKOUT_ATR_PERIOD=14
BREAKOUT_ATR_THRESHOLD=0.01

# === 投資組合 ===
PORTFOLIO=0050:bollinger,006208:bollinger,00878:bollinger,
          2330:ma_cross,2454:ma_cross,
          2881:vwap,2886:vwap,2382:breakout

# === 下單設定 ===
BOLLINGER_POSITION_AMOUNT=15000
VWAP_POSITION_AMOUNT=10000
MA_CROSS_POSITION_AMOUNT=10000
BREAKOUT_POSITION_BUY=1000
BREAKOUT_POSITION_SELL=1000

# === 大盤年線過濾 ===
MARKET_TREND_FILTER=true

# === 金字塔加碼 ===
PYRAMID_ENABLED=true
PYRAMID_TIER1_SHARES=1000
PYRAMID_TIER2_SHARES=2000
PYRAMID_TIER3_SHARES=3000
PYRAMID_TIER2_DROP=0.03
PYRAMID_TIER3_DROP=0.05
```

> 💡 與原版差異：已移除每月預算設定（MONTHLY_BUDGET_*），改為一筆資金初始分配。

## 4. 風險控管

| 防護 | 說明 |
|------|------|
| 大盤年線過濾 | 指數跌破MA200，所有買進訊號暫停 |
| 金字塔加碼 | Bollinger抄底分3梯次進場（1000→2000→3000股） |
| 每日虧損上限 | 單日最大虧損3%（NT$15,000）強制停止 |
| 單筆風險 | 每筆交易最大風險2%（NT$10,000） |
