# 用戶自訂策略功能 - 給使用者

## 概述

本系統提供四組備用策略，您可以自由修改策略邏輯，無需更動主程式 `live_trader_multi.py`（已加密）。

## 檔案清單

| 檔案 | 說明 | 是否可修改 |
|------|------|-----------|
| `user_strategies.py` | 用戶自訂策略定義檔案 | ✅ 可自由修改 |
| `USER_STRATEGIES_GUIDE.md` | 詳細使用說明 | ✅ 參考文檔 |
| `.env.example.txt` | 環境設定範例（含用戶策略參數） | ✅ 參考範例 |
| `test_user_strategies.py` | 策略測試工具 | ✅ 可用來驗證 |
| `integration_test_user_strategies.py` | 完整整合測試 | ✅ 可用來驗證 |

## 快速開始

### 1. 檢查策略載入

```bash
python test_user_strategies.py
```

應該看到類似輸出：
```
✅ 成功載入，找到 4 個策略：
   - g1_strategy_1
   - g1_strategy_2
   - g2_strategy_1
   - g2_strategy_2
```

### 2. 在 `.env` 中使用備用策略

```bash
# 範例 1: 基本使用（使用預設參數）
PC_2330={"strategy":"g1_strategy_1","alloc":20}

# 範例 2: 自訂參數
PC_2330={"strategy":"g1_strategy_1","alloc":20,"fast_period":5,"slow_period":15}
PC_0050={"strategy":"g1_strategy_2","alloc":30,"rsi_period":14,"oversold":25,"overbought":75}
```

### 3. 啟動系統

```bash
python live_trader_multi.py
```

啟動時應該看到：
```
✅ 已載入 4 個用戶自訂策略
```

## 四組備用策略說明

### G1 Strategy 1: 均線交叉策略

**策略代號**: `g1_strategy_1`

**預設邏輯**：
- 快線穿過慢線向上 = 買進
- 快線穿過慢線向下 = 賣出

**可調參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `fast_period` | 5 | 快線週期 |
| `slow_period` | 10 | 慢線週期 |

**`.env` 範例**：
```bash
PC_2330={"strategy":"g1_strategy_1","alloc":20,"fast_period":10,"slow_period":20}
```

---

### G1 Strategy 2: RSI 超買超賣策略

**策略代號**: `g1_strategy_2`

**預設邏輯**：
- RSI < 30 = 超賣，買進
- RSI > 70 = 超買，賣出

**可調參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `rsi_period` | 14 | RSI 計算週期 |
| `oversold` | 30 | 超賣門檻 |
| `overbought` | 70 | 超買門檻 |

**`.env` 範例**：
```bash
PC_2330={"strategy":"g1_strategy_2","alloc":20,"rsi_period":14,"oversold":25,"overbought":75}
```

---

### G2 Strategy 1: 價格動能策略

**策略代號**: `g2_strategy_1`

**預設邏輯**：
- 過去 N 日漲跌幅超過門檻 = 買進
- 過去 N 日跌幅超過門檻 = 賣出

**可調參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `lookback` | 5 | 回看天數 |
| `threshold` | 3 | 漲跌幅門檻（百分比） |

**`.env` 範例**：
```bash
PC_2330={"strategy":"g2_strategy_1","alloc":20,"lookback":10,"threshold":5}
```

---

### G2 Strategy 2: 量價配合策略

**策略代號**: `g2_strategy_2`

**預設邏輯**：
- 價格站穩均線 + 成交量放大 = 買進
- 價格跌破均線 = 賣出

**可調參數**：
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `ma_period` | 10 | 價格均線週期 |
| `volume_ma_period` | 10 | 成交量均線週期 |
| `volume_mult` | 1.5 | 成交量放大倍數 |

**`.env` 範例**：
```bash
PC_2330={"strategy":"g2_strategy_2","alloc":20,"ma_period":20,"volume_ma_period":20,"volume_mult":2.0}
```

## 如何自訂策略邏輯

如果您想修改策略邏輯，請開啟 `user_strategies.py` 檔案：

```bash
vim user_strategies.py
```

每個策略函式都遵循相同格式：

```python
def my_custom_strategy(df: pd.DataFrame, **kwargs) -> pd.DataFrame:
    """
    策略說明

    Parameters:
    - df: 包含 OHLCV 資料的 DataFrame
    - kwargs: 策略參數

    Returns:
    - DataFrame with 'signal' column
        - 1 = BUY (買進)
        - -1 = SELL (賣出)
        - 0 = HOLD (不動作)
    """
    signals = pd.Series(0, index=df.index)

    # 您的策略邏輯...

    return pd.DataFrame({'signal': signals}, index=df.index)
```

修改後記得執行測試：
```bash
python test_user_strategies.py
```

## 混合使用內建策略和自訂策略

您可以同時使用內建策略和自訂策略：

```bash
# 內建策略
PC_0050={"strategy":"bollinger","alloc":30,"window":20,"std_dev":2.0}
PC_2454={"strategy":"keep_wait","alloc":15,"initial_shares":12}

# 自訂策略
PC_2330={"strategy":"g1_strategy_1","alloc":20}
PC_2881={"strategy":"g2_strategy_2","alloc":10}
```

## 常見問題

### Q: 策略無法載入怎麼辦？

A: 檢查 `user_strategies.py` 是否存在，並執行：
```bash
python test_user_strategies.py
```

### Q: 參數沒有生效怎麼辦？

A: 確認參數名稱與策略函式中的 `kwargs.get()` 名稱一致。

### Q: 如何回測自訂策略？

A: 目前內建策略可在 `backtest.py` 中回測，自訂策略請直接在模擬環境中驗證。

### Q: 可以新增第五組策略嗎？

A: 可以！在 `user_strategies.py` 中定義新函式，並加入 `USER_STRATEGY_MAP`，然後在 `.env` 中使用新名稱即可。

當準備加密 `live_trader_multi.py` 時：

1. **確保已整合用戶策略載入邏輯**：
   ```python
   # 載入用戶自訂策略
   try:
       from user_strategies import USER_STRATEGY_MAP
       STRATEGY_FUNCS.update(USER_STRATEGY_MAP)
       print(f"✅ 已載入 {len(USER_STRATEGY_MAP)} 個用戶自訂策略")
   except ImportError:
       print("ℹ️  未找到 user_strategies.py，僅使用內建策略")
   ```

2. **確保 `core/config_loader.py` 已更新**：
   ```python
   STRATEGY_PARAM_KEYS = {
       # ... 原有策略 ...
       "g1_strategy_1": ["fast_period", "slow_period"],
       "g1_strategy_2": ["rsi_period", "oversold", "overbought"],
   ```
