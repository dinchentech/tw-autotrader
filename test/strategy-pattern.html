📋 共同契約

1 def strategy(df: pd.DataFrame, **params) → pd.DataFrame
2 df = df.copy() ← 保護原始資料不被修改
3 計算技術指標 → 新增暫存欄位（MA / BB / VWAP / RSI / ATR …）
4 df['signal'] = 0 ← 初始化
5 df.loc[buy_condition, 'signal'] = 1 ← 買進訊號
6 df.loc[sell_condition, 'signal'] = -1 ← 賣出訊號
7 return df ← 回傳含 signal 欄位的 DataFrame
